## JavaScript Assignment: To-Do List Application

### Objective:
Create a simple to-do list application using HTML, CSS, and JavaScript. The application will allow users to add, delete, and mark tasks as completed.

### Requirements:
1. **HTML Structure**:
    - Create a basic HTML page with a title "To-Do List Application".
    - Add an input field for entering new tasks.
    - Add a button to add the task to the list.
    - Add a section to display the list of tasks.

2. **CSS Styling**:
    - Style the application to make it visually appealing. This includes the input field, buttons, and the task list.
    - Add different styles for completed and pending tasks.

3. **JavaScript Functionality**:
    - Write JavaScript functions to handle the following:
        1. **Add Task**: Adds the task entered in the input field to the task list.
        2. **Delete Task**: Removes a task from the list when a delete button next to the task is clicked.
        3. **Mark as Completed**: Strikes through the task text when it is clicked to mark it as completed.
        4. **Save to Local Storage**: Save the tasks to local storage so that they persist even after the page is refreshed.
